# Help Desk Ticketing System

## Περιγραφή Έργου
Η παρούσα εργασία αφορά την ανάπτυξη μιας διαδικτυακής εφαρμογής τύπου **Help Desk Ticketing System**, η οποία επιτρέπει σε πελάτες (customers) να υποβάλλουν αιτήματα υποστήριξης (tickets) και στην εταιρεία να τα διαχειρίζεται με οργανωμένο τρόπο.

Η εφαρμογή υλοποιείται στο πλαίσιο του μαθήματος **Προγραμματισμός Πλήρους Στοίβας στον Παγκόσμιο Ιστό** και βασίζεται σε τεχνολογίες που διδάχθηκαν κατά τη διάρκεια των διαλέξεων.

---

## Τεχνολογίες που Χρησιμοποιούνται

### Frontend

- Vite
- HTML5
- CSS3
- Bootstrap

### Backend
- Node.js
- Express.js

### Βάση Δεδομένων
- MongoDB (local)

---

## Προαπαιτούμενα
Για την εκτέλεση της εφαρμογής απαιτούνται:
- Node.js (έκδοση 20+)
- npm
- MongoDB (local installation)
- MongoDB Compass (προαιρετικά)

---

## Εκκίνηση Εφαρμογής

### 1. Εκκίνηση MongoDB
Βεβαιωθείτε ότι η MongoDB τρέχει τοπικά στη διεύθυνση:
mongodb://localhost:27017

---

### 2. Εκκίνηση Backend
cd backend
```bash
npm run dev
```

Backend URL:
http://localhost:5001

---

### 3. Εκκίνηση Frontend
cd frontend
```bash
npm run dev
```

Frontend URL:
http://localhost:5173

---
